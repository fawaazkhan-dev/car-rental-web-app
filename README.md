# Car Rental Web Application

A full-stack web application for managing car rentals.

## ⚙️ Requirements
- XAMPP
- MySQL, Apache
- IDE (VS Code, Sublime Text etc.)

## ▶️ How to Run

1. Install XAMPP / WAMP (or any local server)

2. Place project folder in:
   htdocs (for XAMPP)

3. Start:
   - Apache
   - MySQL

4. Import database:
   - Open phpMyAdmin
   - Create a database
   - Import the provided .sql file

5. Open browser:
   http://localhost/car-rental-web-app

---

### Notes
- Ensure database connection settings are correct in the project files

## Features
- User booking system
- Car listing and availability
- Database integration

## Technologies
- HTML, CSS, JavaScript
- MySQL
